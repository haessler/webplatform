{{Summary_Section|An [[apis/webaudio/AudioBuffer|'''AudioBuffer''']] where the output audio data should be written. It will have a number of channels equal to the '''numberOfOutputChannels''' parameter of the [[apis/webaudio/AudioContext/createScriptProcessor|'''createScriptProcessor()''']] method. Script code within the scope of the [[apis/webaudio/ScriptProcessorNode/onaudioprocess|'''onaudioprocess''']] function is expected to modify the [[apis/webaudio/Float32Array|'''Float32Array''']] arrays representing channel data in this [[apis/webaudio/AudioBuffer|'''AudioBuffer''']]. Any script modifications to this [[apis/webaudio/AudioBuffer|'''AudioBuffer''']] outside of this scope will not produce any audible effects.}}
