===Members===

The '''SVGFESpotLightElement''' object has these properties:

*[[svg/properties/limitingConeAngle|'''limitingConeAngle''']]: Gets a value that specifies a limiting cone that restricts the region where the light is projected.
*[[svg/properties/pointsAtX|'''pointsAtX''']]: Gets a value that specifies the x location in the coordinate system established by attribute [[svg/properties/primitiveUnits|'''primitiveUnits''']] on the [[svg/elements/filter|'''filter''']] element of the point at which the light source is pointing.
*[[svg/properties/pointsAtY|'''pointsAtY''']]: Gets a value that specifies the y location in the coordinate system established by attribute [[svg/properties/primitiveUnits|'''primitiveUnits''']] on the [[svg/elements/filter|'''filter''']] element of the point at which the light source is pointing.
*[[svg/properties/pointsAtZ|'''pointsAtZ''']]: Gets a value that specifies the z location in the coordinate system established by attribute [[svg/properties/primitiveUnits|'''primitiveUnits''']] on the [[svg/elements/filter|'''filter''']] element of the point at which the light source is pointing.
*[[svg/properties/specularExponent|'''specularExponent''']]: Gets a value that indicates an exponent value that controls the focus for the light source.
*[[svg/properties/x|'''x''']]: Gets or sets the x-coordinate value.
*[[svg/properties/y|'''y''']]: Gets or sets the y-coordinate value.
*[[svg/properties/z|'''z''']]: Gets a value that indicates the z location for the light source
