|Examples={{Single_Example
|Description=In the following code example, a linear gradient fills a rectangle. The gradient is defined in a definition and fills the rectangle by referencing the gradient. The gradient runs from magenta to cyan.
Copy this sample to a text file and save it with the .html file extension. Run it in Internet Explorer 9 to see the magenta-to-cyan gradient-filled rectangle.

The gradient-filled rectangle will look like this:
|LiveURL=
|Code=

&amp;lt;!DOCTYPE HTML&amp;gt;
&amp;lt;html&amp;gt;
	
  &amp;lt;head&amp;gt;
  &amp;lt;/head&amp;gt;
	
  &amp;lt;body&amp;gt;
    &amp;lt;!-- Inline SVG --&amp;gt;
    &amp;lt;svg width{{=}}"400" height{{=}}"400"&amp;gt;
      &amp;lt;defs&amp;gt;
        &amp;lt;!-- Define linear gradient for magenta to cyan. --&amp;gt;
        &amp;lt;linearGradient id{{=}}"magenta2cyan" &amp;gt;
        &amp;lt;!-- First color is magenta. --&amp;gt;
        &amp;lt;stop offset{{=}}"0%" style{{=}}"stop-color:magenta"/&amp;gt;
        &amp;lt;!-- Second color is cyan. --&amp;gt;
        &amp;lt;stop offset{{=}}"100%" style{{=}}"stop-color:cyan"/&amp;gt;
        &amp;lt;/linearGradient&amp;gt;
      &amp;lt;/defs&amp;gt;
      &amp;lt;!-- Rectangle fill is defined by linear gradient in defs. --&amp;gt;
      &amp;lt;rect width{{=}}"100" height{{=}}"50" x{{=}}"50" y{{=}}"50"
				style{{=}}"fill:url(#magenta2cyan)"/&amp;gt;
				
    &amp;lt;/svg&amp;gt;
  &amp;lt;/body&amp;gt;
&amp;lt;/html&amp;gt;
