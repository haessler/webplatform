|Examples={{Single_Example
|Description=In the following code example, a radial gradient fills a rectangle. The gradient is defined in a definition and fills the rectangle by referencing the gradient. The gradient runs from magenta to cyan, starting in the center with magenta and working outward to cyan.
Copy this sample to a text file and save it with the .html file extension. Run it in Internet Explorer 9 to see the circular magenta to cyan gradient filling the rectangle.

The radial-filled rectangle will look like this:
|LiveURL=
|Code=

&amp;lt;!DOCTYPE HTML&amp;gt;
&amp;lt;html&amp;gt;
	
  &amp;lt;head&amp;gt;
  &amp;lt;/head&amp;gt;
	
  &amp;lt;body&amp;gt;
    &amp;lt;!-- Inline SVG --&amp;gt;
    &amp;lt;svg width{{=}}"400" height{{=}}"400"&amp;gt;
      &amp;lt;defs&amp;gt;
        &amp;lt;!-- Define radial gradient for magenta to cyan. --&amp;gt;
        &amp;lt;radialGradient id{{=}}"magenta2cyan" &amp;gt;
        &amp;lt;!-- First color is magenta. --&amp;gt;
        &amp;lt;stop offset{{=}}"0%" style{{=}}"stop-color:magenta"/&amp;gt;
        &amp;lt;!-- Second color is cyan. --&amp;gt;
        &amp;lt;stop offset{{=}}"100%" style{{=}}"stop-color:cyan"/&amp;gt;
        &amp;lt;/radialGradient&amp;gt;
      &amp;lt;/defs&amp;gt;
      &amp;lt;!-- Rectangle fill is defined by linear gradient in defs. --&amp;gt;
      &amp;lt;rect width{{=}}"100" height{{=}}"50" x{{=}}"50" y{{=}}"50"
				style{{=}}"fill:url(#magenta2cyan)"/&amp;gt;
				
    &amp;lt;/svg&amp;gt;
  &amp;lt;/body&amp;gt;
&amp;lt;/html&amp;gt;

===Remarks===
'''Note'''  In addition to the attributes, properties, events, methods, and styles listed above, SVG elements also inherent core HTML attributes, properties, events, methods, and styles.
The '''radialGradient''' element inherits properties from its ancestors instead of from the element that references  the '''radialGradient''' element.
'''radialGradient''' elements are never rendered directly. They exist  so that you can reference them by  using the &lt;code>fill&lt;/code> and &lt;code>stroke&lt;/code> properties.
The &lt;code>display&lt;/code> property does not apply to the '''radialGradient''' element.  '''radialGradient''' elements are not directly rendered even if the &lt;code>display&lt;/code> property is set to a value other than &lt;code>none&lt;/code>. But  '''radialGradient''' elements are available for referencing even when the &lt;code>display&lt;/code> property on the '''radialGradient''' element or any of its ancestors is set to &lt;code>none&lt;/code>.
