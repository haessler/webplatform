===Remarks===

For a type of '''table''' and a value C &lt; 1, find k such that:
k/''n'' &lt;{{=}} C &lt; (k+1)/''n''
The result C' is given by:
C' {{=}} v<sub>k</sub> + (C - k/''n'')*''n'' * (v<sub>k+1</sub> - v<sub>k</sub>)
If C {{=}} 1 then:
C' {{=}} v''<sub>n</sub>''
For a type of '''discrete''' and a value C &lt; 1 find k such that:
k/''n'' &lt;{{=}} C &lt; (k+1)/''n''
The result C' is given by:
C' {{=}} v<sub>k</sub>
If C {{=}} 1 then:
C' {{=}} v<sub>n-1</sub>
