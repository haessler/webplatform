===Remarks===
If  you do not provide a '''type'''  property, the value of &lt;code>contentScriptType&lt;/code> on the [[svg/elements/svg|'''svg''']] element is used, which  defaults to &lt;code>application/ecmascript&lt;/code>. If a [[svg/elements/script|'''script''']] element falls outside of the outermost '''svg''' element and  you do not provide the '''type''' attribute, the value of &lt;code>contentScriptType&lt;/code> on the '''script''' element  defaults to &lt;code>application/ecmascript&lt;/code>.
