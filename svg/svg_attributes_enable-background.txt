===Remarks===
The optional &amp;lt;x&amp;gt;,&amp;lt;y&amp;gt;,&amp;lt;width&amp;gt;,&amp;lt;height&amp;gt; parameters on the new value are numeric values that indicate the subregion of the container element's user space where access to the background image is allowed to happen. These parameters enable the SVG user agent potentially to allocate smaller temporary image buffers than the default values.
