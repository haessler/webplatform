===Remarks===
The '''type'''   property  specifies the style sheet language of the [[svg/elements/style|'''style''']] element's contents. The style sheet language is specified as a content type (for example, &lt;code>"text/css"&lt;/code>).
If  you do not provide a type, the value of &lt;code>contentStyleType&lt;/code> on the [[svg/elements/svg|'''svg''']] element is used, which  defaults to &lt;code>"text/css"&lt;/code>. If a [[svg/elements/style|'''style''']] element falls outside of the outermost '''svg''' element and  you do not provide the type, the value of &lt;code>contentStyleType&lt;/code> on the '''type''' element defaults to &lt;code>"text/css"&lt;/code>.
